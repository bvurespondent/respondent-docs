---
title: "Retrieve all messages"
description: "get https://api-staging.respondent.io/v1/messaging/messages"
---

Get all messages for a user. Will return messages from both the researcher and the participant.

`conversation.metadata.externalResearcherId` can be used to link the message to an external researcher.

`conversation.metadata.projectId` can be used to link the message to a project.
